西红柿疙瘩汤

发表于 2017-03-26


所需食材：西红柿2个，面粉1小碗，鸡蛋1个，生抽1勺，盐2克，开水2碗，葱花少许，香菜适量，香油少许做法：西红柿洗净去皮。去皮小窍门，用一根筷子插入西红柿底部，放到火上转圈烤几下，这样就能很轻易地撕下西红柿皮了。

2.西红柿用刀削成小块。不要切，因为切容易使西红柿汁流出浪费。

3.锅中放少许食用油，油热后放入葱花，煸出香味。

4.放入西红柿煸炒，加入2克盐，边炒边用铲子切西红柿块，直至出汁。


5.加入一勺生抽，两碗热水。煮开。

6.准备一小碗面粉，把自来水开到最小，一滴一滴接连往下滴的状态，把面粉碗放到下面，用筷子朝一个方向不停地搅拌，搅拌出的疙瘩放一边，再接着搅拌其他干粉。

7.水开后，加入拌好的疙瘩，如果有大块儿的，可以用手揪一揪。

8.取一个鸡蛋打撒，打的过程中加入少许温水，能让煮出的鸡蛋更薄更柔软。

9.开锅后，打入鸡蛋液，关火，点少许香油，加入适量香菜，搅拌均匀即可。

凉拌土豆丝

发表于 2017-03-26


原料：土豆2个调料：香菜1棵、干红辣椒3个、小葱1根、盐、糖、米醋、味精少许、植物油适量

做法：1、 土豆去皮，切薄片； 然后切成细丝；

2、洗去淀粉，并在清水中浸泡一会儿；

3、坐锅烧水，水开后，下入土豆丝焯一下；

4、土豆丝变色后，马上捞出过凉开水，

5、沥干水分备用；

6、添加盐、糖、米醋、味精拌匀，撒上葱花香菜；

7、锅内下油，小火炸香干红辣椒碎；

8、趁热浇在土豆丝上，拌匀即可。

温馨提示：1、土豆丝尽可能切细；2、土豆洗去淀粉后浸泡一会儿，口感会更爽脆；3、土豆焯水时间不宜过长，否则影响口感。

炒金针菇

发表于 2017-03-26

材料：金针菇，剁椒，葱花，酱油，油
做法：
1.锅内水烧开，加入金针菇焯熟。

2.捞出放一个碗里。

3.热锅下油，放入剁椒和少许葱花，可以加一点点酱油，划动几下。剁椒是咸的，酱油不要加天多。

4.趁热，浇在金针菇上，再撒点葱花，成了。吃的时候搅拌均匀，好吃！

酸辣凉拌面

发表于 2017-03-26

用料

意大利面、黄瓜、食盐、陈醋、油泼辣子。

做法

1.黄瓜切成丝备用。
2.意大利面煮熟捞出。
3.将黄瓜丝放入煮好的意大利面中，加入山西老陈醋、少许盐和辣子油即可。

肉末凉拌面

发表于 2017-03-26

用料

猪肉 、青椒、红椒、西芹、面条、生抽、食盐、鸡粉、料酒 。

做法

1.猪肉搅成肉糜，青椒、红椒、西芹切小丁。
2.锅里烧开水，放入面条煮熟后过凉水待用。
3.锅里烧热油，放入肉沫快速炒散，调入生抽和料酒。
4.放入青红椒、西芹丁炒匀，最后放入煮好的面，调入少许盐和鸡粉即可。

芝麻酱凉拌面

发表于 2017-03-26

材料

面条、黄瓜、鸡蛋、豌豆苗、花生、北杏仁、芝麻酱、面酱、生抽、醋、食盐 、蒜 、辣椒油 、橄榄油 、水。

做法

1.黄瓜洗净切丝，芝麻酱澥开：两大勺芝麻酱、一勺面酱、一勺生抽、一勺醋、小半勺盐，搅拌均匀。
2.锅中加1勺橄榄油烧热，鸡蛋打散后倒入热油锅中，切成蛋皮丝，豌豆苗放入开水烫几秒中，大蒜切末，熟花生仁和杏仁切碎备用。
3.煮面条：开水锅里下面条，滚两次水，捞出过冷水、沥干，加入两勺橄榄油拌匀，装在袋子里冷藏一下。

蚝油茄汁拌面

发表于 2017-03-26

材料

面条、胡萝卜 、西红柿 、番茄酱 、酱油 、蚝油、醋、白糖、豌豆(鲜) 、植物油 。

做法

  1. 主料准备好，番茄切小块，胡萝卜切小丁，豌豆焯水后捞出。
  2. 1汤匙蚝油、1汤匙番茄酱、1汤匙酱油、1/2汤匙醋、1/2茶匙白糖调成酱汁备用；锅里烧开水，将面条煮熟。
  3. 将煮好的面捞出过凉水；锅里放油，烧至7、8成熟后放入胡萝卜丁、豌豆翻炒。
  4. 炒至断生后放面条拌炒，随后淋入酱汁，翻炒出香味5.放番茄，拌炒均匀即可。

酸辣凉面

发表于 2017-03-26

材料

面条、西红柿、黄豆芽、虾仁 、黄瓜、生抽、陈醋 、辣椒油、香菜、朝天椒、柠檬汁、芝麻油 、蒜 、白糖 、番茄酱 。

做法

1.鲜虾仁用盐腌制10分钟，西红柿切丁、黄瓜切丝、香菜切碎、辣椒切成圈，黄豆芽摘除根部备用。
2.锅内烧开水，将生面条放入锅内煮至水沸腾，再加入半碗凉水，如此反复两次直至面条煮熟。
3.煮面条的开水，可以放入虾仁及黄豆芽氽烫至熟。
4.生抽、陈醋、白糖、番茄酱、柠檬汁、凉开水(3大匙)、辣椒油、芝麻油、蒜蓉、辣椒圈、香菜碎放在碗内调匀即成凉面料汁。
5.面条放入碗内，表面码上熟虾仁、黄瓜丝、西红柿丁、黄豆芽，再淋上凉面酱汁即可食用。

牛肉酱拌面

发表于 2017-03-26

材料

面条、细香葱 、葱油 、生抽 、酱油、米醋、牛肉酱 、食盐 、白糖 。

做法

1.面条、小香葱和调料。
2.锅内水煮开后，下入面条，煮开添半碗冷水，再次煮开再添半碗冷水，煮开后面条出锅过凉水。
3.充分冷却的面条入事先准备好的冰凉开水中浸泡一下。
4.面条充分沥干后加葱花，以及所有的调料拌匀即可。

花生酱凉拌荞麦面

发表于 2017-03-26

用料

荞麦面、胡萝卜、黄瓜、醋、干辣椒、生抽、花生酱、蜂蜜、水、大蒜、芝麻油、小葱、白芝麻。

做法

1.荞麦面放入开水中，按说明煮2-3分钟至面条熟透，沥干水分在冷水下冲洗至完全冷却，沥干备用。
2.煮面条的时候就可以将所有的调料放在一起，调匀。
3.将冷却的面条和蔬菜放在一起，倒入调料拌匀后用保险膜盖上放冰箱冷藏至少半小时以上。如果放冰箱超过一天，可以在微波炉里加热一下，再调入些生抽，醋和水就会比较滋润。
